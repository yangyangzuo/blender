中上:
    snap(snæp)吸附:
        马蹄铁，物体移动的时候，会有吸附效果，快捷键，shift+tab，如果在移动物体时，发现磕磕盼盼，可能是这个选项打开了





右上：

    相机:
    number0,切换到相机视图
    也可以使用ctrl + number0,来激活一个选中的物体，作为相机,
    例如: 选中灯光，然后ctrl + number0,此时，灯光就被激活为了相机视角，以后每次摁number0,都会以灯光的视角作为相机视角
    如果一个场景中有多个相机，这个很有用





左侧:
    工具栏(快捷键t):


        1.选择(select):
                工作区中的select选项中列出了所有的快捷键
                    全选:               a (当全选了工作区的物体时,会发现有一个物体颜色不一样，这个不一样的物体，会在右边的属性面板中，列出对应的属性)
                    取消所有选择:        aa 或者 alt + a 或者 空白处点击左键
                    挑选:               shift + 鼠标左键选择
                工作区中的左边的菜单栏(快捷键t)中也列出了选择的方式:
                    分别是:
                        select(普通点击选择):               shift+space调出菜单，然后w(此时工具栏激活选择项，并且选择方式是普通点击选择)/通过w切换(此时工具栏激活选择项，并且选择方式是普通点击选择)
                        select box(方形盒子选择):           b/shift+space调出菜单，然后b(此时工具栏激活选择项，并且选择方式是方形选择)/通过w切换(此时工具栏激活选择项，并且选择方式是方形选择)
                        select cirlce(圆形工具选择):        c/shift+space调出菜单，然后c(此时工具栏激活选择项，并且选择方式是圆形选择)/通过w切换(此时工具栏激活选择项，并且选择方式是圆形选择)
                        select lasso(ˈlæsoʊ')(索套选择):    shift+space调出菜单，然后2(此时工具栏激活选择项，并且选择方式是套索选择)/通过w切换(此时工具栏激活选择项，并且选择方式是圆形选择)
                    这4中选择工具，可以通过快捷键w,进行快速切换:
                        如果当前工具栏中激活的是选择工具，则w会在上面的4中选择模式中轮流切换
                        如果当前工具栏中激活的是其他工具，则w会立即激活选择工具(选择工具中保留的上次使用的选择方式) 

                        select box:
                            当激活左边工具栏的方形选择模式后，通过方形圈选，既可选择需要的物体，如果一次选择不满意，再次方选既可
                            如果使用的是快捷键b(增选)，进行box select,如果第一次选择的不满，则可以再次摁b，调出方选功能，此时，摁住shift，然后拖动鼠标左键，就可以把刚才选中的物体取消掉

                        circle select:
                            如果通过快捷键c(增选)(可以通过滚轮，放大和缩小圆形选择范围)，进行圆形选择，则点鼠标左键，(圆形覆盖物体的origin)，既可选中物体，或者点住鼠标左键拖动，当圆形覆盖物体的origin，则物体就可以被选中，如果要取消选择某个物体，可以摁下shift在
                            进行圆形选择

                注意:
                    普通点击选择和方形盒子选择(select box),只要鼠标点击到物体的一部分，或者拉出的方框接触到物体，就会选中物体
                    select circle和select lasso这两种选择工具，仅仅是选择的时候，接触到物体，还不能选中，
                    小圆或者锁套工具必须包含住物体原点(object origin)，才可以选中物体
                    默认情况下，我们是看不到物体的原点的，只有被选中的物体才会显示一个小点(即object origin),但是可以调出来
                    工作区中的右上角菜单中,overlays-object,可以看到，
                    默认勾选了origins，这个表示，当你选择了某一个物体，物体上就显示出原点
                    如果你取消了origins勾选，则即使选中了某个物体，物体上也不会出现原点
                    把origins和origins(all)都勾选上，则工作区中的所有的物体上都会出现origin


        2.cursor(游标):
            shift + c,把游标恢复到坐标系中心点(view->align view->center cursor and view all)
            shift + 右键，更改游标位置
            左键选择，激活工具栏中的游标，然后在工作区，点左键，即可更改游标位置
            shift+space调出菜单，然后空格(此时工具栏激活游标项),然后在工作区，点左键，即可更改游标位置




        3.transform(变换):
            集移动/旋转/收缩于一体,当激活改工具后，选中一个物体，物体内部会出现一个gizmo(gɪzməʊ)active tools，可以平移，旋转，收缩，如果看不到该工具，说明被隐藏了
            在3d viewport中，右上角-> overlays --> active tools,选中该active tools就可以看到物体内容不指针，箭头等
            快捷键(ctrl + `),把鼠标移动到active tools上，会提示(active tool gizmo(gɪzməʊ),shortcut: ctrl `)
