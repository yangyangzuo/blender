(工具栏)toogle toolbar: 快捷键是t

    1.选择(select):
            工作区中的select选项中列出了所有的快捷键
                全选:               a 
                取消所有选择:        aa 或者 alt + a 或者 空白处点击左键
                挑选:               shift + 鼠标左键选择
                反选:               ctrl + i
            工作区中的左边的菜单栏(快捷键t)中也列出了选择的方式:
                分别是:
                    select(普通点击选择):               shift+space调出菜单，然后w(此时工具栏激活选择项，并且选择方式是普通点击选择)/通过w切换(此时工具栏激活选择项，并且选择方式是普通点击选择)
                    select box(方形盒子选择):           b/shift+space调出菜单，然后b(此时工具栏激活选择项，并且选择方式是方形选择)/通过w切换(此时工具栏激活选择项，并且选择方式是方形选择)
                    select cirlce(圆形工具选择):        c/shift+space调出菜单，然后c(此时工具栏激活选择项，并且选择方式是圆形选择)/通过w切换(此时工具栏激活选择项，并且选择方式是圆形选择)
                    select lasso(ˈlæsoʊ')(索套选择):    ctrl+鼠标右键/shift+space调出菜单，然后2(此时工具栏激活选择项，并且选择方式是套索选择)/通过w切换(此时工具栏激活选择项，并且选择方式是圆形选择)
                这4中选择工具，可以通过快捷键w,进行快速切换:
                    如果当前工具栏中激活的是选择工具，则w会在上面的4中选择模式中轮流切换
                    如果当前工具栏中激活的是其他工具，则w会立即激活选择工具(选择工具中保留的上次使用的选择方式)

                    select:
                        当选择工具激活的时候，可以摁下shift，然后通过鼠标左键进行挑选
                        (当多选了工作区的物体时,会发现有一个物体颜色不一样，这个不一样的物体，默认会在右边的属性面板中，列出对应的属性，当然也可以在右边属性面板中手动查看某个物体的属性)
                        当物体被选中以后，在右边的outliner中，对应的元素的图标会有一个背景色(颜色和主题有关)

                    select box:
                        当激活左边工具栏的方形选择模式后，通过方形圈选，既可选择需要的物体，如果一次选择不满意，再次方选既可,
                        如果要取消选中的物体，先ctrl，在框选，即可取消选中的物体

                        如果使用的是快捷键b(增选)，进行box select 
                        取消选中的一部分物体:
                            摁b，调出方选功能，此时，摁住shift，然后拖动鼠标左键，就可以把刚才选中的物体取消掉
                            或者
                            摁b，调出方选功能，然后拖动鼠标中键，就可以把刚才选中的物体取消掉

                    circle select:
                        当激活了左边工具栏的圆形选择的时候，摁下鼠标左键，在软件的最下面的面板就中会提示可以有哪些操作，
                        例如：想放大圆形选择方式的圆圈，此时，按下左键不放，然后，滚动鼠标中键，即可放大/缩小圆形，
                        ctrl，取消其中选中的物体

                        如果通过快捷键c(增选)(可以通过滚轮，放大和缩小圆形选择范围)，进行圆形选择，则点鼠标左键，
                        (圆形覆盖物体的origin)，既可选中物体，或者点住鼠标左键拖动，当圆形覆盖物体的origin，则物体就可以被选中，如果要取消选择某个物体，可以摁下shift在
                        进行圆形选择

                        
            注意:
                普通点击选择和方形盒子选择(select box),只要鼠标点击到物体的一部分，或者拉出的方框接触到物体，就会选中物体
                select circle和select lasso这两种选择工具，仅仅是选择的时候，接触到物体，还不能选中，
                小圆或者锁套工具必须包含住物体原点(object origin)，才可以选中物体
                默认情况下，我们是看不到物体的原点的，只有被选中的物体才会显示一个小点(即object origin),但是可以调出来
                工作区中的右上角菜单中,overlays-object,可以看到，
                默认勾选了origins，这个表示，当你选择了某一个物体，物体上就显示出原点
                如果你取消了origins勾选，则即使选中了某个物体，物体上也不会出现原点
                把origins和origins(all)都勾选上，则工作区中的所有的物体上都会出现origin


    2.cursor(游标):
        3d游标的作用:当你定位好游标后，添加一个新物体，物体会出现在游标所在的位置
        shift + c,把游标恢复到坐标系中心点(view->align view->center cursor and view all)
        shift + 右键，更改游标位置
        左键选择，激活工具栏中的游标，然后在工作区，点左键，即可更改游标位置
        shift+space调出菜单，然后空格(此时工具栏激活游标项),然后在工作区，点左键，即可更改游标位置




    3.transform(变换):
        集移动/旋转/收缩于一体,当激活改工具后，选中一个物体，物体内部会出现一个gizmo(gɪzməʊ)(小玩意/小发明/小物件)active tools，可以平移，旋转，收缩，如果看不到该工具，说明被隐藏了
        在3d viewport中，右上角-> overlays --> active tools,选中该active tools就可以看到物体内容不指针，箭头等
        快捷键(ctrl + `),把鼠标移动到active tools上，会提示(active tool gizmo(gɪzməʊ),shortcut: ctrl `)



    4.move:
        任意移动/特定方向移动/特定面移动,同上，如果gizmo(gɪzməʊ) tool不显示，需要去设置一下
        shift+space调出菜单，然后g(此时工具栏激活move工具),当选择一个物体的时候，左键点击物体/空白地方，拖动，物体就可以任意移动
        点击物体里面的小部件(坐标轴),就可以在特定的轴移动，点击里面的小方块，表示在特定的面移动，例如：点击红色的方块拖动，表示在除了
        红色的x轴以外的轴移动，即：y轴和z轴组成的平面内移动，其实就是在红色块所在的面内移动

        最好使用快捷键g，然后拖动移动，或者根据软件下方的提示，进行具体的移动，例如：想在x轴以外的轴移动/在y轴和z轴组成的面移动，
        先选中一个物体，然后g，然后shift + x,此时，物体上会为显示y轴和z轴组成的面，即在除了x以外的轴上移动

        总结:
            在某个特定的面移动:
            方法1.通过move工具，激活gizmo，然后，点击里面的小方块，小方块所在的面，就是我们移动的时候，物体所在的面
            方法2.快捷键g，然后shift + x/y/z,则物体就可以在对应的轴以外的轴组成的平面内移动
            方法3.通过小键盘切换到前左/右视图，则拖动物体移动的时候，该物体就会在除了x轴以外移动


    移动物体:  
        如果当前的工具栏中，激活的工具，具有移动功能，比如:选择工具中的select(不包含box select,circle select,lasso select),transform,move,
        这些工具中本身带有移动功能，如果这3个工具中的某个被激活，而且在场景中又选择了物体，那么直接在场景中的空白区，拖动鼠标即可移动物体，
        当然点击物体也可以移动，但是，如果物体中的小物件显示出来，则容易误点，
        比如：当前激活了transorm工具，当选择一个物体的时候后,如果想再次点击物体后拖动鼠标移动，容易点到里面的小物件(gizmo),所以最好在物体的外面点击左键，并拖动鼠标来进行移动物体
        如果非要点击物体来进行拖动，1.关闭物体内的小物件(右上的overlay中设置),2,把视角拉近，让物体变大，物体的体积大于小物件的大小即可

        快捷键的方式:g
            选中一个物体后，摁g，然后在软件的最先面会出现提示，你可以直接拖动移动，或者摁下x，然后在x轴移动，等等
            当前左侧工具栏中激活的任何工具都不会对移动造成影响
            例如:
                g,然后s,拖动鼠标，则物体整体等比例缩放
                g,然后x(表示在x轴操作),然后s，拖动鼠标，就可以在x轴进行缩放

        alt + g,把物体移动到坐标系的中心点

    5.rotate:
        选中物体后，快捷键r，在当前视角平面内旋转，x键后在拖动鼠标，则围绕x轴旋转，同理y轴，z轴也一样
        如果连续摁2次r，则物体可以以任意角度，任意面移动
        或者
        shift+space调出菜单，然后r(此时工具栏激活旋转项)

        恢复默认的旋转角度(和坐标轴角度一样),alt + r
    6.scale:
        选中物体后，快捷键s等比例缩放，(s,x,物体在x轴缩放，同理s,y在y轴缩放，s,z，在z轴缩放)如果精确缩放可以摁下shift并同时移动鼠标
        或者
        shift+space调出菜单，然后s(此时工具栏激活缩放项)

        在某个面上缩放/在某2个轴上缩放:
        s,shift + x,拖动鼠标，物体在y轴和z轴上缩放

        恢复初始大小,alt + s

        如果在缩放的时候，发现物理里面的gizmo坐标轴和自己希望的不一样，可以在工作区的中上部分调整坐标系为local，这个时候
        物体缩放的时候，可以按照物体自己本身的坐标来进行缩放

    13.loop cut(环切):
        参考modelling->meshes->editing
        
        cut mesh loop and slide it.(切割网环并滑动)
        激活工具栏的图标:
                鼠标左键点击 
            或者
                shift + spacebar，然后 ctrl + r
            当激活左侧环切工具后，鼠标移动到物体上，则物体上会出现一条环形线框，直接点击鼠标，则会在对应的中心位置
            出现一个环形线框，也可以点击拖拽到合适的位置，再松开鼠标

        使用快捷键:
                直接ctrl + r,此时不会激活左边的工具栏
                鼠标移动到物体上，物体上回出现一个环框，点击一下鼠标左键，环框所在的面确定，
                    (此时可以滚动滑轮或者输入字母上方数字，来控制环框的个数)
                    然后移动位置，点击左键，环框的具体位置确定
                    如果想让环切的位置，正好在物体的正中间，可以直接点右键来确定最终的位置，任意地方点右键，默认会在物体的中间
        https://www.bilibili.com/video/av17804799/?p=12
        


    14.knife:
        切割工具，可以在物体上画出点和线
        快捷键: k
        或者
        shift + space ,k 激活左侧工具栏
        激活工具后，鼠标左键选中起点，然后移动鼠标，移动过程中和物体相交的部位会出现点和线
        激活工具后，移动过程中，鼠标左键松开和不松开有不同的结果
